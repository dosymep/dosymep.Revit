Данная библиотека содержит методы расширения для Autodesk Revit.

## Сборка проекта

Компиляция проекта в папку `bin`
```
nuke compile
```

Локальная публикация проекта
```
nuke publish
```

Сборка и публикация документации
```
nuke doc-compile
```
